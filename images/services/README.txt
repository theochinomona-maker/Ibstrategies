SERVICE IMAGES FOLDER
=====================

Place your service-specific images here:

For the services-template.html, you need:
- service-hero.jpg (Main hero image for the service page - 1920x400px recommended)

For individual service pages, name them accordingly:
- ppe-hero.jpg (for PPE Manufacturing page)
- cleaning-hero.jpg (for Cleaning Services page)
- petroleum-hero.jpg (for Petroleum Supply page)

You can also add other service-specific images here as needed.

Recommended image sizes:
- Hero images: 1920x400px or wider
- Feature images: 800x600px
- Thumbnail images: 400x300px

Make sure to optimize images for web (compress them) to keep file sizes reasonable.
